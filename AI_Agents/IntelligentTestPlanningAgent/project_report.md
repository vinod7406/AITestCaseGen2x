# Project Report: Intelligent Test Planning Agent

## 1. Executive Summary
The **Intelligent Test Planning Agent** is a production-grade AI-powered platform designed to automate the creation of comprehensive test strategies and plans. By integrating directly with enterprise tools like **Jira** and **Confluence**, and leveraging state-of-the-art **LLMs**, the agent reduces the time required for test planning from hours to seconds while maintaining high standards of quality and traceability.

## 3. Core Feature Modules

### 🛠️ Advanced Jira Synchronization
Unlike traditional tools that rely on manual copy-pasting, this agent features deep Jira integration:
*   **Smart JQL Search**: Dynamically fetches the latest user stories or specific sprint requirements.
*   **Deep URL Sanitization**: Users can paste full browser links (e.g., to a specific story); the agent automatically extracts the domain and Issue ID.
*   **Permission Verification**: A "Live Handshake" verify button ensures credentials (Email + API Token) are valid before any work begins.

### 🧠 Verified AI Logical Engine
The system ensures reliability through a "Check-Before-Generate" policy:
*   **Deep Model Verification**: The agent contacts the AI provider (e.g., Ollama or Groq) to verify that the specific model (e.g., `Gemma 3` or `Llama 3.3`) is actually installed and reachable.
*   **Formatted Generation**: Outputs are structured in professional Markdown, covering Functional Flows, Non-Functional requirements, and Edge Cases.

### 📝 Export & Collaboration Suite
*   **Rich Markdown Preview**: Integrated `remark-gfm` support for rendering tables and lists in a "Premium Mode" viewer.
*   **Native Confluence Publishing**: A custom-built bridge converts AI Markdown into **Confluence Storage Format (HTML)**, creating fully formatted wiki pages in personal or team spaces.
*   **Smart Space Detection**: Automatically identifies personal space keys (hashes) from pasted URLs.

### 🗂️ Workspace Management
*   **Curriculum (Templates)**: A built-in library of industry-standard test strategies (Standard, Mobile, Security, Performance).
*   **Persistent History**: Every generated plan is archived locally in a searchable history module for instant recall.
